# Paper

TALLRec: Tuning LLMs for Recommendation
・LLMを推薦データでInstruction Tuningし，ユーザが候補アイテムを好むかをYes/Noで判定する枠組みを示した手法

LoRA: Low-Rank Adaptation of Large Language Models
・LLM本体の重みを固定し，低ランク行列のみを学習することで，少ない計算資源で推薦タスクへ適応するための基盤手法

MeTA-LoRA: Data-Efficient Multi-Task Fine-Tuning for LLMs
・LoRAとメタ学習を組み合わせ，複数タスクの少量データから共有可能な適応能力を学習する手法
